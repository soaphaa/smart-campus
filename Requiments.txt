CS Summative Group Project (3 People)
Project Name: Smart Campus Idle Item Matching & Circulation Platform

Innovation: Exclusive campus-oriented system, different from ordinary second-hand trading platforms
1. Project Overview
This platform is specially designed for teachers and students on campus. It focuses on the reasonable circulation of unused items such as textbooks, daily necessities, sports equipment and digital accessories.
It supports three transaction modes: short-term rental, item exchange and second-hand sale.
The system matches resources intelligently based on users’ major, grade, courses and campus location, solving the problem of waste of student idle resources and high living cost.
2. Core Innovative Highlights
Academic Demand Matching
Automatically recommend textbooks, study notes and review materials according to users’ majors and courses.
Nearby Location Matching
Prioritize matching users in the same building or campus area for easy offline pick-up.
Flexible Rental System
Support weekly, monthly and semester rental instead of one-time purchase.
Idle Item Expiry Reminder
Remind users to release idle seasonal or graduation supplies in time.
Campus Closed-loop Transaction
Safe and convenient trading only inside the school, no external merchants involved.
Data Visualization Analysis
Count popular idle goods and campus resource supply-demand data.

6. Project Introduction (Directly use for report)
This system is a vertical idle resource circulation platform built for campus users. Different from traditional second-hand platforms, it combines academic needs and geographic information to achieve accurate intelligent matching. It provides diversified circulation ways to reduce resource waste and daily consumption cost, and build an efficient and low-carbon campus resource sharing environment.
